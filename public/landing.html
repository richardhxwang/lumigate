Features

Everything you need to manage AI at scale

Enterprise-grade capabilities in a lightweight Docker stack that runs anywhere — from a Raspberry Pi to a production cluster.

Defense in depth

HMAC signatures, ephemeral tokens, per-project RPM limits, IP allowlists, budget caps, anomaly auto-suspend, and MFA for admin access. Every layer independently enforced.

Sub-millisecond overhead

2,300+ QPS with full auth stack active on commodity hardware. All security checks are O(1) in-memory — auth adds zero measurable latency to your AI calls.

Multi-key rotation

Multiple API keys per provider with automatic round-robin, intelligent failover, and per-project key assignment. No single point of failure.

8+ AI providers

OpenAI, Anthropic, Gemini, DeepSeek, Qwen, MiniMax, Kimi, Doubao — all through one endpoint. Add any OpenAI-compatible provider in seconds.

Usage analytics

Real-time cost tracking across all providers with per-project breakdown. 10 currencies, automatic budget enforcement, and exportable usage reports.

Zero-config deployment

One-line install with interactive setup wizard. Auto-TLS via Cloudflare Tunnel, built-in watchdog with <10s crash recovery, atomic writes for zero data loss.

Admin dashboard

Full-featured web UI for managing projects, keys, providers, users, and usage. Role-based access control with root, admin, and user tiers.

Multi-tenant projects

Isolate apps with dedicated projects — each with its own keys, rate limits, budget, model allowlist, and auth mode. Ship AI features without sharing credentials.

Audit trail

Enterprise-grade audit logging for every admin action, key rotation, and configuration change. Automated daily backups with one-click restore.
