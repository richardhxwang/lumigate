Features

Agent Platform, not just a proxy

AI proxy evolved — server-side tool execution, clean streaming, enterprise security. One Docker stack, runs anywhere.

Clean Chat Proxy

POST /v1/chat: clients receive only clean text + file downloads + status events. Zero tool logic on frontend.

Server-side tools

AI outputs tool tags, server intercepts and executes: Excel, Word, PPT, web search, code sandbox. Tags never reach the client.

224 templates

Professional finance templates (DCF, LBO, Black-Scholes, Goldman), business docs, presentations. AI picks the right template automatically.

8 AI providers

OpenAI, Anthropic, Gemini, DeepSeek, MiniMax, Qwen, Kimi, Doubao. API keys or Collector (headless Chrome) — auto-fallback between both.

Defense in depth

HMAC + ephemeral tokens, PII detection, secret masking, command guard, SSRF protection, tool injection prevention. Per-project RPM, budget, model allowlist.

Smart web search

Auto-detects search intent, AI generates keywords, multi-round SearXNG search, results injected as context. Models with built-in search skip SearXNG.

Sub-millisecond overhead

6,400+ QPS on health, 200 concurrent /v1/chat with zero errors. All security checks O(1) in-memory.

Self-healing

Docker healthcheck, auto-restart, macOS watchdog, RPO ≤1s writes, QUIC tunnel. Collector cookies persist across restarts.

LumiChat built-in

Production chat UI: SSE streaming, markdown, file upload, voice input, PocketBase auth, mobile responsive, 10 presets, session history.

How it works

Four steps, zero complexity

Your app sends one POST. LumiGate handles search, tool execution, file generation, and streams back clean results.

1

Send a message

Your app posts to /v1/chat with provider, model, and messages. Standard OpenAI format — nothing new to learn.

2

Auto-enhance

LumiGate detects search intent, generates multi-language keywords, and queries SearXNG. Fresh context injected automatically.

3

Execute tools

AI outputs tool tags, server intercepts and runs them. Spreadsheets generated, code executed, results collected — all server-side.

4

Clean response

Client receives only clean text, file download links, and status events. Zero tool logic needed in your frontend.

For developers

One endpoint.
Three event types.

If your app can read SSE, it can use LumiGate. No SDKs, no function-calling boilerplate, no tool orchestration code.

  • All auth modes: API key, HMAC, ephemeral token, cookie

Built-in intelligence

Smart features,
zero configuration

Every chat session comes with intelligent behaviors built into the platform — no plugins, no extensions.

Smart search

Auto-detects queries needing fresh data. Generates multi-keyword searches, deduplicates results, injects context — all invisible to the user.

Slash commands

/system, /clear, /model, /export, /compact, /search and more. Power-user shortcuts built right into the input bar.

Tips & hints

Contextual tips appear while waiting for responses. Rotating suggestions help users discover features they didn't know existed.

Mid-stream messaging

Send follow-up messages without waiting for the current response to finish. Messages queue and process in order — no lost context.

Session memory

System prompts, presets, and response modes persist across chats. Your preferences are remembered — every session starts right where you left off.

Edit & resend

Click any message to edit and resend. The conversation rewinds to that point — perfect for refining prompts without starting over.
